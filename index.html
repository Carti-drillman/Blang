Introduction

Welcome to the Piet-based esolang! In this language, code is represented as images, where color transitions control the flow of execution.

Tutorial: Creating Your First Piet Program

Follow these steps to create and run your first Piet program:

  1. Step 1: Choose a Drawing Tool - Use an image editor such as MS Paint, GIMP, or an online drawing tool to create your code. Ensure that the tool allows you to use colors precisely.
  2. Step 2: Define Color Transitions - Use the color sheet as a guide to arrange the colors in the image. Each color represents a different command or control flow in your program.
  3. Step 3: Create the Code - Design the image so that the color transitions form the logic of your program. For example, a red to blue transition might increment a variable, while a blue to green transition prints an ASCII character.
  4. Step 4: Save the Image - Save the image as a common format like PNG or JPEG to ensure color integrity.
  5. Step 5: Upload and Run - Use the code editor on this website to upload your image. Click the "Run" button to see the output of your program.

For more complex programs, refer to the Command Reference section.

Command Reference

Color Transition Command
Red to Blue Increment a variable
Blue to Green Output an ASCII character
Green to Yellow Start a loop or conditional jump
Yellow to Red End a loop or conditional block

Color Sheet for Coding

Color Command Description
Red Start or initialize a variable
Blue Increment the value
Green Print an ASCII character
Yellow Loop or conditional jump
Orange Decrement the value
Purple Move left in the program flow
Pink Move right in the program flow
Gray No operation (empty space)
